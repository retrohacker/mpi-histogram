mpi-histogram
===

Generates a histogram of numbers using MPI. Currently the numbers are generated with rand() and have an approximately uniform distribution.

#Usage

### Compile

`mpicc histogram.c -o histogram`

### Running

`mpiexec -n <processes> ./histogram`
